Scanned Document Information Extraction
Challenge

Objective

Extracting tables from PDFs has always been a daunting task, especially tables that have very complex structure and the pattern interpretability is hard.

These financial tables can be a mere gold mine to businesses if they can be extracted and processed accurately.

Solution

We as a team have developed an algorithm to solve the aforementioned problem by detecting tables in PDF documents and parsing them into processable data in different formats of files like which can be downloaded as csv, excel or JSON. This data can further be used to perform analytics to gain important insights and information.

Future Prospects

We are on the verge of automating the complete process of detecting the tables in the PDF files and extracting them as either csv, excel or JSON just by uploading a file and by click of a button.

We are using the YOLO V3 object detection algorithm to detect tables in the PDF files and have annotated our custom dataset and used darknet pre-trained model to detect the tables in any pdf file.

We are working on a customized algorithm that can parse the content which was detected using the trained YOLO algorithm that can detect tables in the PDFs which are converted to images and then store it in a database which can be downloaded into csv, excel or JSON for further analysis.

Also, we plan to build a dashboard that can provide a real time analysis on the processed data.
